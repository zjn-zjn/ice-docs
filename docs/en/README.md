## What is Ice?

Ice is a **lightweight visual Java rule engine** and **business orchestration framework** designed to solve complex business rules and flexible orchestration requirements. As an open-source rule engine, Ice provides a complete visual rule configuration platform, supporting enterprise-level rule management and dynamic orchestration.

### Core Features

- ‚úÖ **Visual Rule Engine**: Intuitive web management interface with visual configuration and real-time preview
- ‚úÖ **High Performance Architecture**: Pure in-memory computation, millisecond response, suitable for high-concurrency scenarios
- ‚úÖ **Flexible Business Orchestration**: Supports multiple orchestration modes (AND/OR/ALL/NONE) for complex business needs
- ‚úÖ **Deep SpringBoot Integration**: Provides Starter for quick integration, supports SpringBoot 2.x and 3.x
- ‚úÖ **Enterprise-Grade Rule Engine**: Production-proven by Agora, Ximalaya, H3C and other enterprises
- ‚úÖ **Zero Learning Curve**: Simple API design, get started in 5 minutes
- ‚úÖ **Zero Database Dependency** (2.0 New): Uses file system storage, no MySQL required
- ‚úÖ **Native Docker Support** (2.0 New): One-click deployment, ready to use
